# GoldenFlow Landing

Standalone Next.js landing page for GoldenFlow CRM.

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Deploy

Push this folder as its own GitHub repository, then import that repository into Vercel.

No environment variables are required for this landing page.

The WhatsApp CTA number is configured in `app/page.tsx` via the `whatsappUrl` constant.
